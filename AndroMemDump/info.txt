
INFO ABOUT RAM FUNCTIONALITIES


1. Dump Entire RAM

This feature allows you to capture the entire contents of RAM. It is available only if the kernel and LiME module have been compiled beforehand.


2. Dump a Single Process

Enables dumping a specific process using Frida, as opposed to dumping the entire memory.


3. Analyze with Volatility2

Allows the .lime file to be analyzed with Volatility2. To perform the analysis, a profile must be created using the .map file generated during kernel compilation. Compared to Volatility3, this version is slower but supports a wider range of plugins.


4. Analyze with Volatility3

Enables analysis of the .lime file with Volatility3. To do this, symbols must be generated using the vmlinux file obtained during kernel compilation.
Compared to Volatility2, this version is faster but supports fewer plugins.


5. Info Functionalities

Provides a detailed explanation of all available RAM functions.


6. Exit

Returns to the previous menu.
